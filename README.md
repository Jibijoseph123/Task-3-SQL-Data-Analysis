# Task 3: SQL for Data Analysis

## Objective
To analyze and extract insights from an ecommerce database using SQL queries.

## Tool Used
- MySQL Workbench

## Dataset
- Self-created ecommerce relational database

## SQL Concepts Covered
- SELECT, WHERE, ORDER BY
- GROUP BY, SUM, AVG
- INNER JOIN
- Subqueries
- Views
- Indexes
- Handling NULL values

## Outcome
Demonstrated the ability to query, analyze, and optimize structured data using SQL.
